Real-time ML-powered fraud detection for WooCommerce orders.

== Description ==

Connects your WooCommerce store to a fraud detection ML API. Every checkout is scored in real-time with explainable results.

Features:
* Real-time fraud scoring at checkout
* Automatic order hold for flagged transactions
* Admin email notifications
* Explainable AI: top-3 risk factors per order
* Configurable thresholds and fallback behavior

== Installation ==

1. Upload the plugin to /wp-content/plugins/woo-fraud-detect/
2. Activate through the Plugins menu
3. Go to WooCommerce > Fraud Detection to configure the API endpoint
4. Ensure the Fraud Detection API is running
